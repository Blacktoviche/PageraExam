How to create your own Exam…. 

First : when you click Create Exam Button you will see a Frame like this

Picture

 

 

you have to fill all fields about the exam ..

Exam Name its just to define name for the XML exam file

in the second panel you you should give some information about you

its optional but its recommended ... you can tell the students about you

be attention once you filled this fields and clicked Build  button

you will not be able to modify this information after filling this fields you should click

Build button to move to the next Frame to make the Questions

 

 

 

 

Step 2:

you should know what to do you will write the question in the area

choose question mode .

if your question has multi answers or single ..

after that you should choose the  answers count that your question has

choose from  2 to 6 ( don't ask me why just 6 question 

because I red about 4000 questions  but I didn't see any question which has more than six answers )

click next button ..

Step 2:

you will see a Frame like this in  the picture below

 

 

 

In the Note area you should write some words to explain the right answer

It will be shown after you finish the exam it could help the student

This note is the answer of question (why this is the right answer) …

After that you must fill all the answers and choose the right answer

In the step one if you chose Multi Answers you will have to choose here

More than one right answer

When you finish this work you should  click on the save button

to save the question in the memory  and the frame will move you to

the question frame it will continue with this moving until your questions finish

( be attention you gave your questions count in the first Step so

the program knows when your questions finish )

there will be message to tell you that your questions is finished and you must click on the

Save Exam to save your exam in XML File in the Exams Folder
